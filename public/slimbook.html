DTD documentation for the SlimBook XML schema.

Use either book or page as root element in your XML document:

    <?xml version="1.0"?>
    <!DOCTYPE ns1:book SYSTEM "slimbook.dtd">
    <ns1:book>
    ...
    </ns1:book>

    <?xml version="1.0"?>
    <!DOCTYPE ns1:page SYSTEM "slimbook.dtd">
    <ns1:page>
    ...
    </ns1:page<

Element Details

ns1:book

This element defines a book containing zero or more chapters. Suitable for type setting a book or creating a set of related web pages.

Declared Attributes

  • #IMPLIED CDATA xmlns:ns1
  • #IMPLIED CDATA xsi:schemaLocation
  • #IMPLIED CDATA xmlns:xsi
